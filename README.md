
# Task Management System for USLS Students

A specialized **Django-based task management system**. This application allows students to track assignments, projects, and exams within a structured, ledger-style interface, featuring dynamic subject-based color coding.

---

## Key Features

* **Create Task:** A streamlined "Add Task" form for adding new academic duties with a modern, boxed UI.
* **Active Ledger:** A real-time table of outstanding requirements.
* **Settled Requirements:** A historical archive for completed tasks with a one-click "Re-open" functionality to move them back to the active ledger.

---

##  Technical Stack

* **Backend:** Python 3.x, Django 5.x
* **Frontend:** Tailwind CSS,
* **Database:** SQLite
